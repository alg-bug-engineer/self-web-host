# 网站升级迭代开发计划 (AI Knowledge Points)

## 🚀 核心目标
将站点从个人实验室/内容展示平台转型为**商业化技术 IP 站**。
- **获取流量**：通过硬核技术文章 (SEO) 和免费 AI 工具引流。
- **建立信任**：通过作品集 (Portfolio) 秀出技术实力（商业级产品背书）。
- **留存互动**：通过 AI 工具箱让用户保持高频访问。
- **最终变现**：引导用户加入“知识星球”或购买专享课程。

---

## 第一阶段：品牌重塑与导航重构 (Phase 1)
*目标：精简结构，确立技术专业度。*

1.  **清理冗余模块**：
    *   从 `Sidebar` 和移动端菜单中移除 `漫画 (Manga)` 和 `实验室 (Lab)`。
    *   隐藏相关路由，防止稀释技术品牌感。
2.  **导航与路由重写**：
    *   重构 `Sidebar.tsx`，将导航调整为：`首页 | 技术专栏 | AI 工具箱 | 我的产品 | 知识星球`。
    *   统一品牌标识为“芝士AI吃鱼”。
3.  **知识星球落地页 (`/planet`)**：
    *   新建 `/src/app/planet/page.tsx`，采用 Landing Page 设计。
    *   包含：痛点分析、会员权益、课程大纲、加入方式（二维码）。
4.  **作品集 (Portfolio) 商业化包装**：
    *   重构 `src/app/portfolio/page.tsx`，突出 Auto-Ops, Cogniflow 等项目的架构图与商业价值。

---

## 第二阶段：转化漏斗与会员权限系统 (Phase 2)
*目标：构建从免费内容到付费会员的闭环。*

1.  **全站引流组件 `PlanetBanner`**：
    *   开发并在所有文章末尾、工具箱显眼位置插入星球引导组件。
2.  **AI 工具箱分层权限**：
    *   扩展 `tools` 数据结构，增加 `isPro` 标识。
    *   实现工具鉴权逻辑：Pro 工具显示“锁定”，点击弹出星球加入引导。
    *   开发轻量级激活码/星球编号校验系统。
3.  **插件化工具基础架构**：
    *   定义插件化接口，支持将绘画、视频生成等交互式工具直接嵌入站点。
4.  **SEO 深度优化**：
    *   动态生成每个页面的 Metadata (TDK)。
    *   配置自动化 `sitemap.xml` 生成。

---

## 第三阶段：管理后台升级与自动化运维 (Phase 3)
*目标：提升运营效率与系统稳定性。*

1.  **高级管理后台 (Admin)**：
    *   **配置中心**：支持后台修改引流文案、上传二维码。
    *   **运维看板**：集成 PM2 状态监控与部署日志。
    *   **流量分析**：集成 Google Analytics / 百度统计数据看板。
2.  **自动化流水线 (CI/CD)**：
    *   完善 GitHub Actions 部署脚本，实现 push 代码后自动构建并平滑重启 ECS 服务。

---

## 第四阶段：商业级细节与交付验证 (Phase 4)
*目标：确保视觉、交互、性能达到商用标准。*

1.  **视觉 UI/UX 调优**：
    *   全面优化 Tailwind 主题，提升跨端适配的美感。
2.  **插件化工具实装**：
    *   上线 1-2 个具体的交互式 AI 插件（如生图、提示词生成）。
3.  **端到端自动化测试**：
    *   使用 **Playwright** 覆盖核心业务路径（文章阅读、工具解锁、星球跳转）。
